# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего римские обозначения цифр от I до IX, получить их десятичные представления.

### Алгоритм решения

Программа принимает от пользователя строку с римскими цифрами, затем с помощью словаря наход соответствия между римскими и десятичными цифрами. Если соответствия нет, вписывается 0, иначе - цифра из словаря. Результат выводиться на экран.

## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.

### Алгоритм решения

Пользователь вводит строку из цифр, программа отфильтровывае нечётные цифры, проверяет, не стала ли последовательность пустой. Если стала - выводиться 0, иначе составляется число из чётных цифр, которые остались после сортировки. Число выводится на экран.

# Задание 2

## Задача 1

### Текст задачи

Указать, файлы с каким расширением встречаются в указанном каталоге
и его подкаталогах реже всего.

### Алгоритм решения

Программа запршивает у пользователя путь к каталогу. После этого она собивает все названия файлов в последовательность, отделяя тип файла от его названия. Одинаковые расширения группируются и программа формирует последовательность из расширений и количества файлов с этими расширениями. Программа находит минимальное число файлов с каким-то расширением, затем более часто встречающиеся расширения отфильтровываются, потом происходит упорядочивание по имени расширения и получается итоговая последовательность. Выводиться число расширений и само расширение.
